VueJS Directives

v-if

Purple box is visible below if variable isVisible is true.

If variable is false with v-if, nothing shows up in the DOM. In most cases you would use v-if, because you either want something to render or not.

v-if box

If variable is false with v-show, element is visible in the DOM with a style of "display: none". V-show would be used to be more performant for a toggle button, etc.
