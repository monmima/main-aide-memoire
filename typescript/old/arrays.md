# Arrays

## Array of numbers

	let ids: number[] = [1, 2, 3, 4, 5];

## Array of any values

	let arr: any[] = [1, true, "Hello"];

## Tuples

With a tuple, you can specify specific values for specific incides.

	let person: [number, string, boolean] = [1, "Brad", true];

## Tuple array

	let employee: [number, string][]
	employee = [
		[1, "Brad"],
		[2, "John"],
		[3, "Jill"],
	]
