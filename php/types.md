# Data types

- strings
- integers
- floats
- booleans
- arrays
- objects
- NULL
- Resource (reference to a function; not an actual data type)

## Examples

    $string = "To be or not to be";

    $integer = 10;

    $float = 10.50;

    $isBoolean = true;

    // stands for no value
    $null = null;

    echo $string;
